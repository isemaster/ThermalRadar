# TERMO1 Radar — v0.1.0

Android-приложение для парапланеризма. Анализирует микрораскачку крыла через акселерометр телефона, определяет направление на термик, отслеживает крутку и помогает центрировать ядро.

## Возможности

### Базовая функциональность
- **Детекция термиков** — полосовой фильтр Баттерворта 0.25–2.5 Гц, RMS-окно, zero-crossing frequency detector
- **Радар** — направление на термик с привязкой к компасу, кольца дальности 50/100/125/150м
- **Вариометр** — барометрический, с адаптивным сглаживанием, звук Brauniger IQ
- **Голосовые подсказки TTS** — направление + дистанция, 45° сектора, русский язык
- **Вибрация** — при смене статуса (ПОИСК → ТЕРМИК РЯДОМ → НАБОР)
- **Калибровка наклона** — запоминает угол крепления телефона на груди
- **Симуляция** — 75-секундный демо-режим полёта
- **Тестовый режим** — 6 шагов проверки датчиков с real-time фидбеком

### Новое в v0.1.0

- **🔥 Детекция крутки + центровка** — автоопределение по гироскопу (18°/с, 4 с), золотая надпись «крутим термик» после 540°, секторный анализ N/E/S/W, голосовые подсказки после каждого круга: *«ядро на ветер»*, *«ядро по ветру»*, *«Ядро север/юг/восток/запад»*
- **💨 Ветер по дрейфу спиралей** — трекинг GPS-центров спиралей, стрелка от края радара к центру + скорость ветра в м/с
- **📝 Журнал событий (Events.csv)** — голосовые подсказки, появление/исчезновение блипов, начало/конец крутки, обновления ветра, центровка
- **⏱ Монотонное время** — все dtMs на `elapsedRealtime` (без скачков от NTP/перевода часов)
- **📊 Расширенный Samples.csv** — +4 колонки: `gpsFixAge`, `gpsAccuracy`, `vario`, `detectStatus` (30 полей)
- **🎯 TH_SUSPECT 0.005g → 0.020g + SNR>3** — порог поднят, SNR-фильтр отсекает ложные срабатывания в снижении
- **🛑 Кнопка СТОП с подтверждением** — диалог «Остановить запись? Да/Нет», случайно не прервать
- **📡 GPS всегда включён** — координаты пишутся в лог независимо от настроек
- **👁 Слепой режим (карман)** — яркость экрана 15%, показывает варио/высоту/время
- **☀️ Солнечный режим** — переключатель в настройках, макс. яркость + контрастный HUD
- **📘 Документация** — [VERSION_010.md](VERSION_010.md) — полное описание релиза

## Требования

- Android 8.0+ (API 26)
- Акселерометр (обязательно)
- Барометр (опционально — для вариометра)
- Магнитометр (опционально — для компаса)
- GPS (опционально — для лога)

## Установка

Скачать APK из [релизов](https://github.com/isemaster/ThermalRadar/releases) и установить на телефон.

```bash
adb install ThermalRadar-v0.1.0.apk
```

## Сборка

```bash
./gradlew assembleDebug
```

## Структура проекта

```
app/src/main/java/com/termo1/radar/
├── MainActivity.java           — главный экран, сенсоры, рендеринг
├── core/
│   ├── SignalProcessor.java    — фильтр Баттерворта + RMS + ZC
│   ├── SimulationManager.java  — симуляция 75с
│   └── ThermalDetector.java    — детекция/верификация термиков
├── flight/
│   ├── CirclingManager.java    — детекция крутки, центровка, ветер (новое)
│   ├── FlightStateMachine.java — автостарт по высоте
│   └── BlindFlightMode.java    — слепой полёт (gravity tilt compensation)
├── sensors/
│   ├── SensorController.java   — регистрация датчиков, компас, калибровка наклона
│   └── VarioManager.java       — баровариометр с адаптивным alpha
├── gps/
│   └── GpsManager.java         — FusedLocationProvider
├── logging/
│   └── LogManager.java         — 50 Гц, ZIP, события, elapsedRealtime
├── model/
│   └── ThermalBlip.java        — модель blip на радаре
└── ui/
    ├── RadarRenderer.java      — отрисовка радара Canvas
    ├── SettingsActivity.java   — экран настроек
    ├── UiManager.java          — HUD: варио, высоты, статус
    └── VarioSoundManager.java  — звук AudioTrack (Brauniger IQ)
```
