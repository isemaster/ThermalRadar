# 🪂 TERMO1 Radar — Инструкция пилота

**Версия:** 0.2.15  
**Назначение:** Android-прибор для поиска и обработки термиков в парапланеризме  
**Совместимость:** Android 8.0+ (любой телефон с барометром)

---

## 1. Что это и зачем

TERMO1 Radar — это **вариометр + радар термиков** в вашем телефоне. Он помогает:

- **Находить термики** по микрораскачке крыла — раньше, чем варио покажет набор
- **Центрировать термик** — показывает направление и расстояние до центра
- **Оценивать ветер** — направление и скорость на вашей высоте
- **Записывать полёт** — CSV + IGC лог для анализа

### Главная идея

> Прибор анализирует микровибрации крыла в фазе набора и подсказывает, **куда довернуть**, чтобы остаться в ядре термика. Для новичка это как «второй пилот», который говорит: «Отлично, держи левее» или «Термик уходит, доверни правее».

## 2. Экран — что есть что

```
┌─────────────────────────────────────┐
│  ↑ +2.3 м/с         [⚙] [✕]       │  ← варио (скорость набора)
│  ПОИСК / ТЕРМИК / НАБОР             │  ← статус
│                                     │
│         [ N ]                       │
│          │                          │
│   [W]────+────[E]   ○ — термик     │  ← радар (150 м)
│          │          ← — снос        │
│         [ S ]       ◉ — центр       │
│                                     │
│  50м ○ 100м ○ 125м ○               │  ← кольца дальности
│                                     │
│  Центр: 45° 120м                    │  ← цель для доворота
│  ↙ Снос: 12° вправо                 │  ← куда сносит
│                                     │
│  ВЫС 1200м MSL  +340м AGL          │  ← высота
│  ВЕТЕР 230° 3.2 м/с                │  ← ветер на вашей высоте
│  [КАЛИБР]    [ЗАПИСЬ]              │  ← кнопки
└─────────────────────────────────────┘
```

### Элементы радара

| Элемент | Что означает |
|---------|-------------|
| **Синие кольца** | 50 м, 100 м, 125 м — дистанция от вас |
| **Белый трек** | Ваш GPS-след (последние 5 минут) |
| **Жёлтая точка** | Подтверждённый термик (направление + расстояние) |
| **Зелёная зона** | Сектор с лучшим подъёмом (крутитесь туда) |
| **Стрелка сноса** | Куда вас сносит ветром относительно термика |
| **Красный центр** | Ядро термика (расчётное или учебное) |
| **Зелёный маркер** 🟢 | Точка входа в термик |
| **Красный маркер** 🔴 | Точка выхода из термика |

### Статусы

| Статус | Что значит | Что делать |
|--------|-----------|------------|
| **ПОИСК** | Летите прямо, термиков рядом нет | Летайте змейкой, ждите |
| **СПИРАЛЬ** | Замечена слабая раскачка | Присмотритесь, возможно термик рядом |
| **ТЕРМИК** | Раскачка усиливается, но набор ещё мал | Начинайте крутку в сторону раскачки |
| **НАБОР** | Уверенный набор, вы в термике | Центрируйте по радару |
| **СНИЖЕНИЕ** | Пикирование или сильный нисходящий | Ищите новый термик |
| **ВАРИО ТЕРМИК** | Термик по варио (подтверждён набором) | Точно в ядре, крутите |

## 3. Как пользоваться в полёте

### 3.1. Перед полётом

1. **Закрепите телефон** на грудной обвязке или в кармане штатива
2. Проверьте, что барометр работает (на `Settings` → `Barometer` видно давление)
3. Убедитесь, что GPS ловит (зелёная иконка)
4. **Покрутите телефон «восьмёркой»** — это калибрует магнитометр
5. Нажмите **«КАЛИБР»** для калибровки наклона телефона (он должен быть неподвижен 2 секунды)

### 3.2. Полёт — стадия 1: ПОИСК

- Летите змейкой по участку
- Прибор слушает микрораскачку крыла (0.25–2.5 Гц)
- Когда статус сменится на **«СПИРАЛЬ»** — рядом есть термик
- Когда на **«ТЕРМИК»** — пора начинать крутку

### 3.3. Полёт — стадия 2: ЗАХОД В ТЕРМИК

Когда статус **«ТЕРМИК»**:

1. Посмотрите на радар — жёлтая точка показывает **направление и расстояние** до термика
2. Доверните в ту сторону на 3–5 секунд
3. Если варио показывает набор — начинайте крутку
4. Если нет — ещё 3 секунды и ищите другой

> **Совет:** На начальном этапе не гонитесь за точностью. Даже если вы промахнулись на 50 м, термик всё равно вас поднимет. Радар нужен для второго-третьего круга, когда вы точно хотите остаться в ядре.

### 3.4. Полёт — стадия 3: КРУТКА В ТЕРМИКЕ

Статус **«НАБОР»**:

- Прибор показывает **сектор лучшего подъёма** (зелёная область на секторной диаграмме)
- Голосовые подсказки (если включены):
  - *«Северо-восток, по ветру»* — доверните на СВ
  - *«Центр термика 45 градусов, 120 метров»* — до центра столько
- **Следите за сносом** — если ветер сносит, центр термика смещается по ветру (показано на радаре)

> **Важно:** Не пытайтесь стоять в одной точке. Ветер сносит и вас, и термик. Доворачивайте по подсказкам.

### 3.5. Полёт — стадия 4: УХОД

Когда набор прекратился:

- Статус **«СНИЖЕНИЕ»** или **«ПОИСК»**
- На радаре остаются отметки термиков (живут ~30 секунд)
- Зелёные маркеры 🟢 — где вы зашли
- Красные 🔴 — где вышли
- Если видите, что вышли из термика в правильном направлении — можете повторить

### 3.6. Тренировка: слепой режим

В настройках можно включить **«Слепой полёт»**:

- Экран гаснет (яркость 15%)
- Голосовые подсказки работают в полную силу
- Телефон в кармане, вы летите **только на слух**

Рекомендуется для опытных пилотов для отработки чутья термиков.

## 4. Настройки

### Основные

| Параметр | Значение по умолч. | Что делает |
|---------|-------------------|-----------|
| Термик-порог, м/с | 0.5 | Чувствительность варио-термика |
| Blind Mode | выкл | Слепой полёт (только голос) |
| Голосовые подсказки | вкл | Направления ветра/термика голосом |
| Солнечный режим | выкл | Макс. яркость экрана для солнца |
| Ночной режим | выкл | Тёмная цветовая схема |
| Воздушная скорость | 9.5 м/с | Ваша средняя скорость (для расчёта ветра) |

### Режимы цветовой схемы

| Схема | Для кого |
|-------|----------|
| Стандартная | Обычные условия |
| Высокий контраст | Яркое солнце |
| Ночная | Вечерние/ночные полёты |

## 5. Логирование

### CSV-лог

- Частота: **25 Гц**
- Содержит: время, акселерометр (X/Y/Z), гироскоп, магнитометр, давление, GPS, варио, курс
- Позволяет проанализировать полёт в Excel или Python

### IGC-лог

- Стандартный IGC-формат (совместим с XCSoar, SeeYou, Leonardo)
- Частота: **1 Гц** (GPS + давление)
- H-записи: пилот, прибор, дата
- B-записи: время, координаты, высота GPS, высота барометрическая
- L-записи: события (вход/выход из термика)
- CRС32 G-запись

### Как начать лог

1. Нажмите **«ЗАПИСЬ»** на экране
2. Лог пишется в папку `Documents/TERMO1/`
3. Для остановки — нажмите кнопку «Стоп» (с подтверждением)
4. Лог автоматически запускается при обнаружении полёта

## 6. Расшифровка статусов

```
ПОИСК ──(раскачка >порога)──→ СПИРАЛЬ
СПИРАЛЬ ──(подтверждение)──→ ТЕРМИК
ТЕРМИК ──(набор >1 м/с)──→ НАБОР
НАБОР ──(снижение < -1 м/с)──→ ПОИСК
```

| Состояние | Критерий | Время |
|-----------|----------|------|
| ПОИСК → СПИРАЛЬ | Активность в раскачке + 0 Гц пик > SNR | ~3 секунды |
| СПИРАЛЬ → ТЕРМИК | Подтверждение роста амплитуды | ~1 секунда |
| Прямой → Крутка | Угловая скорость > 4°/с | ~4 секунды |
| Крутка → Прямой | Угловая скорость < 4°/с | ~5 секунд |

## 7. FAQ

### ❓ Радар показывает термик, а варио молчит
Нормально. Микрораскачка крыла появляется за **2–5 секунд** до начала набора. Доверните в сторону термика — варио покажет набор.

### ❓ Компас дрожит на столе
Нормально. В воздухе дрожания нет. Если всё же беспокоит — покрутите телефон «восьмёркой» перед полётом.

### ❓ При резком повороте компас отстаёт
Так и задумано. Фильтр сглаживает курс, чтобы стрелка не дёргалась. Отставание ~150 мс — для параплана незаметно.

### ❓ Что означает зелёный сектор на круговой диаграмме?
Сектор с **наилучшим подъёмом** за последние 30 секунд крутки. Доворачивайте в этот сектор.

### ❓ Как часто обновляется радар?
Радар рисуется 30 раз в секунду. GPS-трек обновляется раз в секунду. Термики — по мере обнаружения.

### ❓ Телефон греется
TERMO1 Radar использует GPS + датчики + экран постоянно. На солнце телефон может нагреваться. Используйте **blind mode** для экономии батареи и уменьшения нагрева.

### ❓ Батарея быстро садится
Расчёт: 3–4 часа непрерывной работы на телефоне 2019 года с батареей 3500 мА·ч. Рекомендуется внешний аккумулятор.

### ❓ IGC-лог не открывается в SeeYou
Убедитесь, что файл имеет расширение `.igc`. Если не открывается — проверьте H-записи (пилот, прибор). Если проблема остаётся — откройте issue на GitHub.
