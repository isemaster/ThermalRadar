# TERMO1 Radar — Инструкция по использованию

**Версия:** 0.0.5

---

## 1. ЧТО ЭТО

TERMO1 — Android-приложение для парапланеризма. Анализирует микрораскачку крыла через **акселерометр телефона**, определяет **направление на термик** и показывает его на радарном экране. Голосовые подсказки и вибрация помогают пилоту не отвлекаться от полёта.

Телефон крепится к парапланеристу (на грудь, в карман harness'а или в карман куртки для слепого режима).

---

## 2. ТРЕБОВАНИЯ К ТЕЛЕФОНУ

| Датчик | Обязательно? | Для чего |
|--------|-------------|----------|
| Акселерометр (TYPE_ACCELEROMETER) | **Да** | Детектор + компас + tilt |
| Барометр (TYPE_PRESSURE) | Нет | Вариометр (без него — только GPS высота) |
| Магнитометр (TYPE_MAGNETIC_FIELD) | Нет | Компас (без него — оси телефона) |
| Гироскоп | Нет | Только журналирование |
| GPS | Нет | Скорость, курс в логе |

**Минимальный:** акселерометр (встроен во все Android).

---

## 3. ЗАПУСК

1. Установить APK
2. Телефон автоматически переходит в полноэкранный режим
3. Яркость: средняя (50%), в полёте при записи — максимальная
4. GPS не включается (экономия энергии)

---

## 4. ЧТО НА ЭКРАНЕ

```
  [×]                            [⚙]
  ┌───────────────────────────────────┐
  │            ПОИСК                  │ ← статус
  │            +1.2                   │ ← вариометр (цифра, без "м/с")
  │                                  │
  │         ┌── N ──┐                │
  │    W ───│   ●   │─── E          │ ← радар (N = север)
  │         │   ⬤   │                │    ● пилот, ⬤ термик
  │         └── S ──┘                │
  │    50м 100м 125м 150м           │ ← кольца дальности
  │                                  │
  │[СТАРТ]        [СТОП]            │ ← кнопки
  │пишем лог                        │ ← красным в полёте
  │±0.12 м/с² | 1.35 Гц | 045°      │ ← дебаг строка
  │Крепление: 35° | Крен: 30°       │ ← углы наклона
  │45м                      1500м    │ ← AGL слева, MSL справа
  │       00:12:34                   │ ← время полёта (без "+")
  │       12:34:56                   │ ← системное время
  └──────────────────────────────────┘
```

### Поля:

1. **Вариометр** — вертикальная скорость, м/с. Зелёный = набор, красный = снижение.
2. **Статус-пилл**: ПОИСК / ПОДОЗРЕНИЕ / ТЕРМИК РЯДОМ / ВНУТРИ ТЕРМИКА
3. **Радар**: вращается по компасу, жёлтый круг = термик
4. **Дебаг строка**: турбулентность (м/с²), частота (Гц), направление (°)
5. **Углы наклона**: Крепление — запомненный угол, Крен — текущий
6. **Высоты**: AGL (над землёй) слева, MSL (над морем) справа
7. **Время полёта** и **системное время**

### Кнопки:

| Кнопка | Действие |
|--------|----------|
| × | Выход |
| ⚙ | Настройки |
| СТАРТ | Начать запись лога (тап). Долгий тап (>0.6с) — калибровка |
| СТОП | Остановить запись лога (всегда видна, тусклая без записи) |
| ТЕСТ | Пошаговый тест датчиков (6 шагов, под высотами) |

## 5. РЕЖИМЫ ДЕТЕКЦИИ

| Статус | Уровень (g) | Дистанция | Что происходит |
|--------|------------|-----------|----------------|
| ПОИСК | < 0.005 | > 150 м | Термиков нет |
| ПОДОЗРЕНИЕ | 0.005–0.020 | 75–150 м | Есть сигнал |
| ТЕРМИК РЯДОМ | 0.020–0.080 | 37–75 м | Стабильное направление |
| ВНУТРИ ТЕРМИКА | > 0.080 | < 37.5 м | Пилот в зоне подъёма |

Направление блокируется после 2+ периодов стабильного сигнала.

---

## 6. ГОЛОСОВЫЕ ПОДСКАЗКИ (TTS)

Сектора по 45° относительно курса:

| Сектор | Фраза |
|--------|-------|
| Front ±22.5° | «Термик спереди 50 метров» |
| Front-Right 22.5–67.5° | «Термик спереди справа» |
| Right 67.5–112.5° | «Термик справа» |
| Front-Left (−22.5…−67.5°) | «Термик спереди слева» |
| Left (−67.5…−112.5°) | «Термик слева» |
| Rear (112.5–180°) | Молчок |

- Дебаунс 8 секунд
- Полная громкость (STREAM_MUSIC max)
- Включается/выключается в настройках

---

## 7. ВИБРАЦИЯ

Короткий импульс (300 мс) при смене статуса:
- ПОИСК → ПОДОЗРЕНИЕ
- ПОДОЗРЕНИЕ → ТЕРМИК РЯДОМ
- ТЕРМИК РЯДОМ → ВНУТРИ ТЕРМИКА / НАБОР

Включается/выключается в настройках.

| Набор | Звук |

## 10. КАЛИБРОВКА НАКЛОНА

**Зачем:** Телефон может быть наклонён на груди пилота под разными углами (0–90°). Калибровка запоминает этот угол, чтобы корректно детектировать горизонтальные колебания.

**Как сделать:**
1. Повиснуть в подвеске неподвижно
2. Настройки → кнопка **Калибровка** (или долгий тап на СТАРТ)
3. Подождать 2 секунды — телефон запомнит угол
4. На экране появится: «Крепление: 35° | Крен: 32°»

**В полёте:** угол крепления статичен, крен меняется (спираль, наклон). Пороги детектора корректируются по углу крепления.

## 11. ЛОГИРОВАНИЕ

### Ручной режим
СТАРТ → запись 50 Гц. СТОП → ZIP-файл.

### Автоматический режим
FlightStateMachine: высота +10 м за 10 с → старт. Высота ±2 м за 5 мин → стоп.

### Нарезка
Каждые 10 минут — новый ZIP-чанк:
- `Start{время}.zip` — первый чанк
- `Flight{время}.zip` — серединные
- `Finish{время}.zip` — последний

### CSV формат (27 колонок)
```
dtMs,gpsSpeed,gpsHeading,gpsLat,gpsLon,gpsAlt,
ax,ay,az,gx,gy,gz,mx,my,mz,
pressure,pitch,roll,heading,
thermalAngle,thermalStrength,thermalDist,thermalSource,
snr,noiseFloor
```

GPS колонки первыми (1 Гц данные, остальное 50 Гц).
1 час полёта ≈ 1.5 МБ в ZIP.

## 12. СЛЕПОЙ РЕЖИМ (КАРМАН)

Включается в настройках.

- Экран почти погашен (brightness 0.01)
- Чёрный фон, только надпись «Слепой полёт»
- Детекция через BlindFlightMode (gravity-based tilt compensation)
- Только TTS, без визуала
- WakeLock остаётся (CPU не спит)

Требование к пилоту: лево-право (ось X) должна быть ориентирована верно.
Наклон может быть любым (0–150°).

---

## 13. СИМУЛЯЦИЯ

Настройки → **Симуляция 75 с**.

Сценарий:
- 0–5 с: полёт на север, снижение −1 м/с
- 5 с: термик справа (100 м)
- 5–15 с: поворот к термику
- 15–50 с: спираль, набор +2.5–3.5 м/с
- 50–75 с: выход, затухание

По окончании — возврат в настройки.

---

## 14. ТЕСТИРОВАНИЕ ДАТЧИКОВ

Кнопка **ТЕСТ** — 6 шагов с обратной связью:

| Шаг | Действие | Проверка |
|-----|---------|----------|
| 1 | Неподвижно 5 с | Калибровка шума |
| 2 | Качать влево-вправо | Акселерометр |
| 3 | Качать вперёд-назад | Акселерометр |
| 4 | Повернуть на 90° влево | Компас |
| 5 | Крутить 360° с наклоном | Компас + акселерометр |
| 6 | Трясти 3–5 Гц | Полоса фильтра |

После прохождения — диалог отправки лога.

```
  │
  ▼

  │
  ▼

```

## 16. СОВЕТЫ ПИЛОТУ

1. **Перед полётом:** откалибровать наклон (Настройки → Калибровка)
2. **Телефон крепить жёстко** к harness'у
3. **Жёлтый шарик** = куда лететь
4. **Голос** говорит направление + метры
5. Долгий тап на СТАРТ — сброс калибровки
6. «ЛЕТАЙ ЗМЕЙКОЙ» — 3 раза сигнал затухал, меняйте курс
7. Логи отправляйте из настроек для анализа
